Accessibility Service Usage

AppCage uses the Android AccessibilityService API to support app blocking and focus session features selected by the user.

The AccessibilityService is used only to:

  • Detect when a blocked app is opened
  • Prevent access to distracting apps during active focus sessions
  • Help users stay focused and productive

AppCage does NOT use AccessibilityService to:

  • Collect personal or sensitive data
  • Read messages, passwords, or typed text
  • Monitor browsing activity
  • Track user behavior outside the app
  • Record screen content or user input

All Accessibility features are optional and must be manually enabled by the user in Android Accessibility Settings. Users can disable the service at any time.
